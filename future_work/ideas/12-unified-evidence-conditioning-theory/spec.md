# Idea 12 — 证据条件化的统一理论:Kernel / Score / Sampler 三位一体

- 目标会议:TMLR(或 ICLR 2028)| 层级:C(理论纲领,低成本长线)
- 一句话:把"外部证据信号 e 如何进入推荐学习"统一为一个倾斜目标分布族的三种估计器 — 改腐蚀核(我们的 AAAI'27)、改 guidance(CFG 族)、改负采样(去偏 CL 族)— 刻画三者的偏差-方差-安全性三角,给出"何时该在哪一侧条件化"的处方定理。

## 1. Gap

1. 三条线在文献里互不引用:kernel 侧(本组)、score 侧(CFG-rec 2409.10494、A2G-DiffRec、A-CFG)、sampler 侧(Debiased-CL、SRNS、硬负例挖掘)— 但形式上都是对同一倾斜分布 p_e(x) ∝ p(x)·exp(η·e(x)) 的操作,**无人写下这个公共对象**;
2. 各线的失效模式(负例污染 / 过度引导 / 修正偏差)散落各处,没有统一的风险语言 — 我们的 TV 界、碰撞命题、φ 定价是现成的统一件;
3. 处方缺失:实践者面对"我有个内容信号"时,选注入位置纯靠门派 — 一张有定理支撑的决策表是真实缺口。

## 2. 方法核心(理论论文结构)

```text
统一对象:倾斜目标 p_e;三种实现 = (i) 前向腐蚀核替换(改数据分布,训练时),
  (ii) 采样期 score 组合(改估计器,推理时),(iii) 负例分布修正(改损失权重,训练时)
三定理骨架:
  T1(安全性):只有 (i) 存在精确归约(e→0 ⇒ 恒等宿主);(ii)(iii) 的残余偏差下界
  T2(偏差-方差):固定倾斜强度 η 下三者的估计方差排序与样本量/批规模的关系
  T3(处方):给定 (U_e, 服务约束, 重训成本) 的最优注入位置分区图
实证:一个 2×3 合成 + 2 个真实数据集的最小对照(борrow Idea 08 的表),不打 SOTA
```

## 3. 决定性检验(理论论文的"Gate-0")

1. T1 的 (ii)(iii) 残余偏差下界必须非平凡(不是 0 也不是 ∞)— 若 score 侧也能构造精确归约(比如 w→0 平凡成立…需要定义"条件化生效时的安全性",即固定非零 η 下的回退能力),定义要重铸;**先花一周把定义钉死再动笔**;
2. T3 分区图给出的处方在 Idea 08 的实验表上方向正确;
3. 与 RLHF/DPO 的倾斜分布文献(Bradley-Terry 视角,PreferGrow 论文自己引的)接通 — 相关工作即卖点。

## 4. 风险 / 价值

风险:统一框架论文易被评"重新包装已知内容"— 防线是 T1/T3 必须有非显然内容,且实证表是文献中不存在的直接对照。价值:即使不独立成文,T1-T3 是 01/08/10 三篇的共享理论骨架,写作成本一次摊三处。
