# Idea 07 — 跨模态一致性分数的零点校准(CLIPScore 的地板问题)

- 目标会议:CIKM 2027(全文)+ 开源工具 | 层级:C
- 一句话:把 AAAI'27 的"原始一致性地板 0.93-0.95 随嵌入几何漂移"发现推广到**跨模态一致性分数**(CLIPScore 式图文对齐分):证明未校准分数在跨数据集/跨编码器比较中系统性误导模态加权,给出置换零点校准修正,横扫修复一批融合方法。

## 1. Gap

1. 多模态推荐大量使用原始跨模态相似度做模态加权/清洗依据(Trustworthy-MM 2602.00730 的 Sinkhorn 整流、REVEAL 2606.09082 的 reweighting、MuSICRec 2602.07208 的 ID 门),**没有一个对照"随机配对能得多少分"的零基线** — 我们在单模态一致性上已实证这个地板高得离谱且随几何漂移;
2. CV/NLP 社区知道 CLIPScore 有校准问题(温度、各向异性),但推荐场景的**用户历史级聚合**(历史均值向量再相似)使地板问题被平方放大 — 无人量化;
3. 修正是通用件:任何"用图文一致性决定信任谁"的方法都可直接换上校准分。

## 2. 方法核心

```text
每 (数据集, 编码器对) 冻结三种 null:随机物品对 / 置换图文配对 / 长度匹配伪历史
校准分 s̃ = (s − μ_null) / σ_null(结构与 ũ 同)
理论小件:各向异性嵌入下 E[随机对相似度] 的下界与主成分谱的关系(解释地板从哪来)
```

验证面:选 3 个公开可复现的模态加权方法(候选:REVEAL、BM3/FREEDOM 家族、Trustworthy-MM),把其内部一致性信号替换为校准版,统一协议重测。

## 3. 决定性实验

1. **误导性实证**(核心,纯离线 1 周):跨 10 数据集 × 3 编码器,原始一致性分的数据集间排序与校准后排序的秩相关 — 若 >0.9(排序基本不变),"误导"不成立,idea 死;我们的单模态先验证据(ML1M null 0.931 vs Steam 0.942 但残差差 5 倍)强烈暗示会低于 0.5;
2. 修复增益:≥2 个方法在 ≥2 个数据集上换校准分后显著变化(方向不限 — 变好是贡献,变坏且能解释也是贡献);
3. 地板来源消融:去各向异性(whitening)后地板降幅 vs 置换校准的等效性。

## 4. 风险

- 复现他人方法的工程成本(选官方开源的);
- 若结论是"校准后几乎没差别",转短文/负结果轨,成本已花在可复用的 null 管线上(并入 Idea 02)。

## 5. 时间线

2026-11 决定性实验(02 的管线就绪后顺手做)→ 2027-05 CIKM 截稿。
