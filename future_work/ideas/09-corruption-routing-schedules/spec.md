# Idea 09 — 腐蚀路由调度:偏好该向哪个模态邻域褪色、何时褪

- 目标会议:NeurIPS 2027 | 层级:C(高风险高新颖)
- 一句话:把"腐蚀分布"从静态混合升级为**时变路由调度** w_m(t):扩散早期褪向行为先验(安全、探索),晚期褪向模态语义邻域(信息量大、险),调度形状本身成为设计空间 — 离散扩散推荐的"噪声调度设计"对应物。

## 1. Gap

1. 连续扩散有丰富的噪声调度研究(cosine/EDM/SNR 加权),**离散推荐扩散的"proposal 调度"完全空白**:PreferGrow/LLaDA-Rec/StageCF(2605.05165 定制了腐蚀过程但无模态、无时变路由)都是全程单一 proposal;
2. 我们的碰撞命题给了调度设计的第一性原理:**碰撞险区(高效用锚)应放在 ᾱ_t 大的步(早期,腐蚀概率低)还是小的步(晚期)** — 有明确可推导的权衡,现在无人知道答案;
3. 时变 proposal 的理论地位:每步仍 rank-one ⇒ 逐步闭式保留,但乘积不再 telescope(不同 Π_m 不对易)— **这打破我们自己的命题 1 闭式**,需要新理论(分段常数调度下的两段闭式、或对易化近似 + 误差界),失败风险真实。

## 2. 方法核心

```text
P_t = α_t·I + (1−α_t)·Π_{w(t)},  Π_{w(t)} 的行 = Σ_m w_m(t)·c^m + w_0(t)·p̃_core
调度族(冻结参数化):w(t) = softmax(θ·schedule_basis(t)),basis ∈ {常数, 线性, 两段}
理论目标:两段调度(t<τ 用 p_core,t≥τ 用混合锚)的 q_t 闭式 + 目标碰撞质量的时间分解
  → 推导"碰撞最小的 τ 放置"与 (U_ds, ᾱ 轨迹) 的关系,给可检验预言
```

## 3. 决定性实验

1. **两段调度的方向检验**(核心):Beauty 上 {全程 core, 全程混合, 早 core 晚锚, 早锚晚 core} 四臂 — 理论预言的最优段序必须胜出且与碰撞分解一致,否则理论-实验断裂,idea 降级;
2. τ 扫描曲线与理论预测曲线的形状吻合度;
3. 险区数据集上"晚锚"是否如碰撞理论预言地放大伤害。

## 4. 风险(诚实:本组合最高)

- 非对易乘积的理论可能只剩数值验证 → NeurIPS 变 SIGIR;
- 调度收益可能是二阶小量(单段已 |Δ|~0.006 量级)→ 决定性实验 1 若四臂差 < 种子噪声,立即砍,沉没成本仅 1 周 GPU;
- 与 05 的层级腐蚀思想部分重叠 → 若 05 先跑通,本 idea 并入其"调度节"。

## 5. 时间线

2027-02 决定性实验(等 01 的多锚基建)→ 2027-05 NeurIPS 截稿。
