# Idea 05 — 腐蚀面效用感知的多模态语义 ID Codebook

- 目标会议:KDD 2027(8 月周期)| 层级:B
- 一句话:RQ-VAE 语义 ID 的 codebook 学习加一个**腐蚀面正则** — 阻止"强预测正例的内容方向"被编进将用于离散扩散/生成腐蚀的 code 前缀,把反转律做进 tokenizer。

## 1. Gap

1. 语义 ID 是生成式推荐主战场(RaG 2606.25496、SSRLive 2606.06970、MoToRec 2602.11062、TokenMinds 2606.25147),codebook 全部以**重构/对齐/均衡**为目标训练,对下游"这些 code 会被当负例/腐蚀目标使用"完全无感;
2. LLaDA-Rec(2511.06254)已把离散扩散跑在 SID 上 — 它的腐蚀(mask/替换 SID token)恰好继承我们证明过的 false-negative 风险:**语义相近前缀 = 高效用方向 = 腐蚀污染**;SIDInspector(2606.10375)诊断 aliasing 但没有效用维度;
3. 我们的 U_ds 目前定义在原始 embedding 上 — 推广到"code 前缀效用" U_prefix 是自然但无人做过的一步。

## 2. 方法核心

```text
U_prefix(ℓ) := P[ 真实下一物品与历史共享前 ℓ 层 code | 流行度匹配负例不共享 ]   # 前缀级效用
codebook 训练目标 = RQ-VAE 重构 + λ·前缀效用整形:
   浅层(将被激进腐蚀的层)推向低 U_prefix(行为中性、类目级),
   深层(少被腐蚀)允许高效用(个性化区分度)
```

- 直觉:让"腐蚀常打的层"天然安全(褪向类目邻居 = hard negative),"腐蚀少打的层"承载预测力;
- 附带理论:SID 离散扩散的逐层腐蚀等价于块结构 proposal,层级 φ 即块级门控 — 与 01 的核框架同族,命题可迁移。

## 3. 决定性实验

1. **前缀效用可整形性**(核心,1 周):固定重构损失预算下,λ 扫描能否把浅层 U_prefix 压低 ≥30% 而深层保持 — 不可整形(重构与效用强耦合)则 idea 死;
2. LLaDA-Rec 复现 + 换我们的 codebook:掩码扩散生成质量在高效用类目(ML1M 型)是否显著改善;
3. SIDInspector 式诊断:aliasing/覆盖不因正则恶化。

## 4. 资产杠杆 / 风险

U_ds 估计器改造为前缀版即可;M3L 与 Amazon 图文 bank 直接喂 RQ-VAE。风险:LLaDA-Rec 复现成本(无官方细节时降级为自建小型 SID 扩散);工业 SID 论文竞争密度极高,必须靠"效用整形"这个独有轴,不打规模。

## 5. 时间线

2026-10 决定性实验 → 2027-02 KDD 一轮(或 2027-08 二轮)。
