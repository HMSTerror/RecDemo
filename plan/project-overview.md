# PreferGrow AAAI-27 paper closeout

The active objective is to close the evidence and wording gaps for the AAAI-27 submission without selectively replacing the frozen legacy Table 2 values or overstating seed-100 observations. The authoritative r7 execution state is `issues/2026-07-12_14-41-00-aaai27-r7-atomic-repair.csv`; older ledgers remain historical evidence rather than the current controller.

The paper separates three evidence generations. Legacy `U_ds` is a four-domain train-only discovery descriptor. RISK-03 EPE/PNE@10 directly measures observed next-positive exposure on frozen train transitions. The r6a/r7 pilot uses a dated `phi_R` evidence-retention scale whose implemented direction for the observed values is increasing in EPE, so it is not presented as proof that high exposure mechanically closes the gate.

Theory and empirical evidence remain separate. Exact reduction is a kernel-level statement at `g=0`; nonzero-gate total-variation bounds apply only to the proposal and one-step forward transition rows. The initial E1 step-0 optimizer-ownership mismatch was repaired, after which revision-scoped R12 passed 2,986 designated in-memory comparisons; this still does not prove standalone checkpoint replay or recommendation-metric equivalence. r6a provides only seed-100 precursor observations and explicit-zero fallback sanity checks because it lacks valid anchor artifacts and a RISK-08 exit. r7 is the non-post-hoc same-root repair and must not enter a performance claim until its fourteen active artifacts and original RISK-08 decision exist.

SASRec is the current common-contract external baseline. DiffuRec remains in related work and evaluator-audit provenance but is excluded from the confirmatory comparison. Model selection uses validation only; test metrics were logged during development, so the manuscript does not call test an untouched final holdout.
