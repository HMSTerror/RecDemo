### 1.2 文献 gap(2026-07 扫描,来源见 `future_work/04-landscape-2026.md`)

离散扩散生成式推荐浪潮(2025Q4-2026H1)的调度/腐蚀设计现状:

1. **LLaDA-Rec**(arXiv:2511.06254):离散扩散并行生成语义 ID——调度均匀,无证据定价;
2. **DiffGRM**(arXiv:2510.21805,WWW'26):按模型置信度任意顺序解码 SID 位——自适应信号是**模型内生置信度**,非训练前可算的证据统计量,且无归约保证;
3. **自适应掩码调度**(arXiv:2601.19501,2026-01):发现"SID 各位重要性不同、均匀掩码有害",提出**启发式**噪声调度——正是我们的空位:无定价机制(什么信号决定调度?)、无理论保证(调度偏离均匀时目标是否仍合法?退化时是否还原?);
4. **掩码扩散推荐理论**(arXiv:2511.23021,UT Austin):分析视角,但未触及证据条件化;
5. dLLM 侧(LLaDA→LLaDA-2.0 100B→统一多模态):调度不变性只在**时间轴**上被讨论(MDLM 式),**位置/证据维度的非均匀性**理论空白。

**归纳的空位:没有任何工作用"训练前可算的证据效用统计量"为离散扩散的腐蚀/掩码调度定价,并给出"定价信号为零时精确还原宿主"的保证。** 我们拥有的独占资产恰好是:U_ds 估计量(冻结、已在 5 个数据集有生产数值)、fallback-safe 归约理论套件、零点校准方法论、全套单卡实验基建。

### 1.3 从失败到假设的推理链(v3 的科学动机,全部可引用工件)

1. 用户级门控失败了**两次**(ũ 构念错误 + FOLLOWUP-09 用户级不反转)⇒ 证据效用不是用户级信号;
2. 数据集级 U_ds 反转 4/4 成立 ⇒ 证据效用**是**数据集级信号;
3. 但 φ(U_ds) 开门后的收益不稳(Steam 量级未达、ASO 直接 MISS)⇒ **常数门(对整条腐蚀轨迹均匀作用)是过钝的执行器**——同样的证据,在腐蚀早期(高 α,轻噪)与晚期(低 α,重噪)的价值不应相同;
4. 假设 H3:**证据效用应为腐蚀调度定价**——决定"何时(哪个噪声档)、多大程度向文本锚点褪色",而非"是否常数开门"。2601.19501 在 SID 位置维度独立观察到"均匀有害",与 H3 在时间维度同构。

## 2. 方法(v3:证据定价调度)

### 2.1 记号

沿用 v2:词表 Ω(M+1 态含伪物品 ⋆),宿主 proposal `p_core=softmax(p1)`,文本锚 `c_h`,秩一核 `Π_p(i,j)=p(j)`,生成元 `Q_p=Π_p−I`,前向 `P_{σ,p}=e^{σQ_p}=αI+(1−α)Π_p`(α=e^{−σ})。v2 门 `g=g_max·φ(U_ds)·clamp(ũ,0,1)` 为轨迹常数。

### 2.2 实例化 (a):时间依赖门(本仓库可跑,BRIDGE-04 的对象)

```text
g(t) = g_max · φ(U_ds) · s(t),   t = 腐蚀步/噪声档
p_u,t = p_core(⋆)e_⋆ + (1−p_core(⋆))[(1−g(t))·renorm(p_core) + g(t)·c_h]
```

- **设计决定 D1(冻结)**:主配置**去掉用户因子 ũ**(依据 §1.3 第 1 条),ũ 作为消融臂保留;
- **设计决定 D2(冻结)**:调度形状族 S = {const, early-fade, late-fade},以离散化 Beta 密度实现(峰值归一化);具体 (a,b) 参数在 BRIDGE-02 理论备忘中、**任何跑动之前**冻结(与 φ 的 7.2 同款纪律);
- **设计决定 D3(冻结)**:s(t)≡1 时 v3a 精确还原"v2 去 ũ"配置;φ(U_ds)=0 时任意 s 均精确还原宿主(见 P1′)。伪物品质量、g_max、φ(0.70,0.10)、τ0、k 全部沿用冻结值,不新增可学习模块。

### 2.3 实例化 (b):位置依赖掩码率(可移植叙事,SIGIR'27 论文的一般化框架)

掩码扩散语义 ID 场景:位置 i 的掩码率 `w_i ∝ f(Î_i, φ(U_ds))`,其中 Î_i 为 train-only 的位置区分度统计量(U_ds 的位置分解)。(b) 不在本仓库跑,作为论文框架的第二实例与 2601.19501 的直接对话面;其理论对应 P4′。

### 2.4 实现面(BRIDGE-03 的改动清单,预估)

1. `model/text_side.py`:门计算接受噪声档参数 t(现为轨迹常数;改动面 = proposal 构造入口签名);
2. `graph_lib.py` 四个核调用点(≈:1323 起):前向腐蚀/score-entropy 权重/反演比率改用 §3 L1′ 的**有效 proposal p̄_t**闭式(关键:不是逐步重算矩阵指数);
3. `losses.py`/`sampling.py`:透传 t;
4. 测试:`s≡1 ⇔ v2-无ũ` 等价套件 + `φ=0 ⇒ 宿主` 等价套件(g≡0 套件的直接扩展);
5. 复用不动:U_ds 工件加载(hash 校验)、null 曲线、冻结 bank、tmux 编排/快照/重试 helper。

## 3. 数学命题候选(证明策略与风险分级)

> 记号:步 i 的存活率 α_i,proposal p_i(由 g(t_i) 决定),ᾱ_n=∏α_i。

**L1′(时变秩一前向的闭式,enabling lemma,难度:低,置信:高)**
对任意 proposal 序列 p_1..p_n:∏_i[α_i I+(1−α_i)Π_{p_i}] = ᾱ_n I + (1−ᾱ_n)Π_{p̄_n},其中 p̄_n = Σ_i w_i p_i,w_i = (1−α_i)∏_{j>i}α_j / (1−ᾱ_n)。
*证明策略*:归纳 + 秩一核吸收律 Π_a Π_b = Π_b。
*意义*:时间依赖门下前向边缘仍是"存活-否则重采样"形式,**有效 proposal 是路径上各 proposal 的显式凸组合**;score-entropy 权重与解析反演比率保持闭式可用。这是 v3a 全部理论的地基,也是它区别于"任意调度魔改"的合法性来源。

**P1′(调度级归约保证,难度:低)**
φ(U_ds)=0 或 s≡0 ⇒ ∀t: p_u,t=p_core ⇒ p̄_t=p_core ⇒ 前向/权重/反演/平稳分布逐对象等于宿主。
*证明策略*:老命题 6 逐 t 应用 + L1′。s≡1 时还原 v2-无ũ(D3 的形式化)。

**P2′(时变核下训练目标与反演的合法性,核心定理,难度:中,真功夫所在)**
陈述目标:以 p̄_t 替换 p 后,(i) 去噪 score-entropy 损失保持其闭式与变分界地位;(ii) 非齐次链的时间反演比率保持"宿主公式换 proposal 项"的形式。
*证明策略*:(i) SEDD 框架已容许 σ(t)Q 的时间缩放;非齐次 Q_{p_t} 情形用 L1′ 的边缘闭式直接重推 DSE 权重(边缘形式不变 ⇒ 权重公式结构不变);(ii) 非齐次马氏链反演的标准公式(逐时刻用 q_t 边缘比率)+ L1′ 边缘的显式性。
*风险*:反演步的 proposal 项出现的是 **p̄_t 而非 p_t**(路径累积效应)——若推导确认此点,这本身是一个值得写的发现(采样器必须用累积有效 proposal,naive 逐步替换是错的);若推导发现权重结构改变,则 v3a 需限制在"分段常数调度"子族,风险受控。

**P3′(TV 界的调度版,难度:低-中)**
TV(p̄_t, p_core) ≤ Σ_i w_i g(t_i) =: ḡ_t(**有效累积门**,可解释为"路径上的证据暴露量");行级 TV(P^{0→t}_{v3}(i,·), P^{0→t}_{core}(i,·)) ≤ (1−ᾱ_t)·ḡ_t。
*证明策略*:凸组合的 TV 次可加 + 老引理逐项。ḡ_t 给出论文级的可视化对象(证据暴露曲线)。

**P4′(位置重要性单调性,实例化 (b) 的理论,难度:中-高,风险:代理模型假设强度)**
陈述目标:独立位置代理模型下,若位置信息量 I_i 单调、FN 风险随 U 单调,则代理目标 J(w)=Σ_i[benefit(w_i,I_i) − λ·FN(w_i,U)] 的最优分配 w* 对 I_i 单调。
*证明策略*:交换论证/重排不等式。
*诚实定位*:形式化 2601.19501 的经验观察;含金量取决于代理与真实训练动力学的距离,论文中作为"设计原则的形式化"而非普适定理。

**P5′(方差缩减,期权,难度:高,可能不成立)**
效用对齐分配下加权目标的梯度方差是否 ≤ 均匀分配。仅作探索性附录;失败即弃,不入正文。

**证明工作量预估**:L1′+P1′+P3′ ≈ 3-4 天(BRIDGE-02 前半);P2′ ≈ 1-2 周(BRIDGE-02 主体);P4′ ≈ 1 周(可与实验并行);P5′ 不设预算。
