# Hypothetical-success paper package

This directory is a scenario-planning manuscript, not an evidence artifact and not the active submission. It assumes that every preregistered experiment eventually satisfies its success criterion, while leaving every result cell blank. Blank is not zero. No text or table may be copied into the active paper until the corresponding dated artifacts exist and the claim-evidence gate is updated.

Core confirmatory baselines: SASRec, Caser, GRURec, DiffRec, matched PreferGrow host, and the proposed method. BERT4Rec is an extended baseline. DreamRec, PreferDiff, and DDSR are shown only in a separately gated block and require official-code common-contract audits before local comparison. DiffuRec remains excluded.
